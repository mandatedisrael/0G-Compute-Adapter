Crush

CLI

Overview

Crush is a terminal-based AI coding tool by Charm. Connect it to the 0G Compute Adapter for cost-effective AI-assisted coding.

Installation

# macOS
brew install charmbracelet/tap/crush

# npm (cross-platform)
npm install -g @charmland/crush

# Arch Linux
yay -S crush-bin

Setup

Edit your Crush config file:

  • macOS/Linux: ~/.config/crush/crush.json
  • Windows: %USERPROFILE%\.config\crush\crush.json
{
  "providers": {
    "0g": {
      "base_url": "http://localhost:8000/v1",
      "api_key": "your-0g-api-key"
    }
  }
}

Launch Crush, then press Ctrl+P and select "Switch Model" to pick a 0G model.

Capabilities

Once configured, Crush enables code generation, debugging, file operations, and technical Q&A through AI-enhanced terminal workflows — all at 0G network prices.
