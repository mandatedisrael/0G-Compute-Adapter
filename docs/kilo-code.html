Using 0G Compute Adapter with Kilo Code

Kilo Code is a VS Code extension for AI-assisted coding. Connect it to the 0G Compute Adapter to use decentralized models at a fraction of the cost.

Setup
Connect Kilo Code to 0G in 3 steps
Step 1

Install Extension

Launch VS Code, open the Extensions Marketplace, search for "Kilo Code", and click Install.

Step 2

Select Provider

Choose "OpenAI Compatible" as the API provider.

Step 3

Configure

Enter the following values in the settings fields:
